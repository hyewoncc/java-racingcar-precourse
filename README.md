# 🏁 미션 - 자동차 경주 게임  

## 구현 기능 목록  

- 🎮 입력
  - 자동차 이름 목록 입력  
    - 이름이 5자를 초과할 시 예외  
  - 경주 실행 횟수 입력  
    - 숫자가 아닐 시 예외  
  - 입력이 없을 시 공통 예외  
  - 예외 발생 에러 메시지 출력 후 재입력 요청  


- 🏎️ 자동차
  - 랜덤값에 따라 전진  
  - 자신의 이름과 진행 상태 출력  


- 💻 게임 제어 
  - 입력된 이름의 자동차 생성 
  - 입력된 횟수만큼 자동차 전진 실행  
  - 우승자 판단
  - 우승자 출력 
    - 우승자는 중복될 수 있음
